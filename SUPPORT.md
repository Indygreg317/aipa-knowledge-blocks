# Support This Project

Thank you for taking time to look at this work.

These projects are being built in the open to help create more transparent, auditable, and human-centered AI systems.

The goal is simple:

**Build useful open frameworks that help people structure uncertainty, verify decisions, and create better AI systems.**

---

## Why Support Matters

Independent open-source work takes time.

Support helps free up time for:

- developing Knowledge Block schemas
- expanding working examples
- improving documentation
- building reference implementations
- creating educational materials
- supporting AIPA-aligned open standards
- keeping the work accessible to the public

This work is being shared openly so others can learn from it, build on it, test it, challenge it, and improve it.

---

## Current Focus

Current development priorities include:

- Knowledge Blocks
- deterministic decision frameworks
- receipted / auditable execution records
- AI governance and transparency patterns
- quantum-output decision examples
- humanitarian and public-benefit use cases
- AIPA roadmap development

## Transparency

Support does not buy control over the project.

The intent is to keep this work open, useful, and aligned with ethical AI development.

This project is early-stage and experimental. It is not medical, legal, financial, or deployment-ready advice.

---

## Mission

The long-term goal is to help build AI systems that are:

- transparent
- auditable
- accountable
- human-reviewable
- beneficial to people

Support helps move that mission forward.
